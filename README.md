# msa-general-service
## 목표 
### 설계
- 의존성 주입과 모듈 분리, 육각형 아키턱처를 활용한 유연한 아키텍처 설계 
  - 역할 구성: 요청 접수, 로직 처리, 요청 의뢰 
  - 모듈 구성: 각 역할에 대한 인터페이스 모듈과 구현체 모듈 분리  

### 동시성 
- 자바 동시성 관련 다양한 시도에 대해 k6로 성능 분석 
  - 자바 모니터락 
  - Lock 객체 
  - 세마포어 객체 
  - 동시성 지원 자료구조
